## 时钟模块

## 信号和电压

### 音频信号  
通常在10Vpp范围内（从-5V到+5V）。  

### 控制电压  
由LFO和ADSR等调制源产生，LFO通常从-2.5 V到+2.5 V（5 Vpp），ADSR从0 V到+8 V。

### 触发电压  
触发器、门或时钟信号，启动过程或功能，是矩形信号，典型电压电平为0/+5 V。在触发应用程序的情况下，通常是用于触发事件的信号的上升边缘。
